# String Matching Algorithms

This project implements three string matching algorithms in Python.

## Algorithms

- Naive String Matching
- Knuth-Morris-Pratt (KMP)
- Rabin-Karp

## Features

- Finds all occurrences of a pattern in text
- Counts character comparisons
- Performance comparison on random text

## Technologies

- Python 3
## Output
The program prints:
- Match positions
- Number of comparisons
- Performance comparison table
